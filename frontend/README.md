
  # SaaS Inventory Management Prototype

  This is a code bundle for SaaS Inventory Management Prototype. The original project is available at https://www.figma.com/design/4FEQ1TjIvatitNkUaK3Vql/SaaS-Inventory-Management-Prototype.


##Instalaciona de node js  para npm
winget install OpenJS.NodeJS.LTS

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  












  Extras

  # Verificar el instalador de Windows
winget --version

# Buscar e instalar Docker Desktop de forma automática
winget install Docker.DockerDesktop --accept-source-agreements --accept-package-agreements

# Actualizar el subsistema de Windows para Linux (Requisito de Docker)
wsl --update



# Instalar dependencias del proyecto, controlador de base de datos y lector de variables .env
npm install
npm install mysql2 dotenv

# Instalar los tipos de TypeScript para el entorno de desarrollo
npm install -D @types/node
